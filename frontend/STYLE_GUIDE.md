# StockSense Style Guide

## Color Palette

### Backgrounds
- **App Background**: `#F6F9FF` (Light Blue/Grey)
- **Card Background**: `#EAEAF7` (Soft Lavender)
- **Glass Overlay**: `rgba(255, 255, 255, 0.6)`

### Accents (Pastel)
- **Blue**: `#8EC5FF`
- **Green**: `#C6FFD9`
- **Pink**: `#FFD6E0`
- **Lavender**: `#F7F3FF`

### Sentiment
- **Fear**: `#FFD6E0` (Pink)
- **Neutral**: `#F3F4F6` (Grey)
- **Greed**: `#C6FFD9` (Green)

### Text
- **Primary**: `#1F2937` (Dark Grey)
- **Secondary**: `#6B7280` (Medium Grey)

## Typography

**Font Family**: Inter, System UI

| Role | Size | Weight |
|------|------|--------|
| Page Title | 28px | 700 (Bold) |
| Card Title | 18-20px | 600 (SemiBold) |
| Body | 14-16px | 400 (Regular) |
| Small/Meta | 12px | 400/500 |

## Spacing & Radius

- **Card Radius**: `24px`
- **Panel Radius**: `16px`
- **Standard Padding**: `24px`
- **Grid Gap**: `24px`

## Effects

- **Glass Shadow**: `0 8px 32px 0 rgba(31, 38, 135, 0.07)`
- **Soft Shadow**: `0 4px 20px rgba(0, 0, 0, 0.05)`
- **Blur**: `backdrop-filter: blur(12px)`
